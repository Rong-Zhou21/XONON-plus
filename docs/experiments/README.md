# Experiments

Use this directory for small, Git-tracked experiment notes. Keep raw outputs in
ignored directories such as `exp_results/`, `logs/`, and `videos/`.

Recommended structure:

- `experiment_log.md`: one-line index of all runs and ideas.
- `YYYY-MM-DD_short_name.md`: detailed note for one experiment or ablation.
- `template.md`: copy this file when starting a new experiment note.

Each note should include enough information to reproduce the run from a fresh
checkout: commit hash, branch, command, environment, changed files, metrics,
raw artifact paths, and conclusion.
